# POI Collector

跨平台 POI（兴趣点）数据采集工具，支持从多个主流地图平台批量采集、管理和导出 POI 数据。

![Platform](https://img.shields.io/badge/Platform-Windows%20|%20macOS-blue)
![Tech](https://img.shields.io/badge/Built%20with-Tauri%202.x%20+%20React-orange)

## 功能特性

### 🗺️ 多平台采集
- **天地图** - 国家测绘地理信息局官方地图服务
- **高德地图** - 阿里巴巴旗下导航地图
- **百度地图** - 百度旗下地图服务
- **OpenStreetMap** - 开源免费地图 (无需 API Key)

### 🧩 瓦片地图下载
- **多平台支持** - 谷歌、百度、高德、腾讯、天地图、OSM、ArcGIS
- **多种地图类型** - 卫星图、街道图、地形图、混合图等
- **区域选择** - 支持矩形框选和多边形绘制两种模式
- **多级别下载** - 自定义缩放级别范围 (1-20级)
- **并发控制** - 可调节下载线程数 (1-32)
- **断点续传** - 支持暂停/继续，失败瓦片可重试
- **格式转换** - 支持将下载的瓦片合并为 MBTiles 格式
- **实时进度** - 显示下载速度、进度百分比、预估大小

### 📍 地区管理
- 内置全国省/市/县三级行政区划数据
- 支持多选地区批量采集
- 快速搜索定位目标地区

### 🔑 API Key 管理
- 每个平台支持配置多个 API Key
- 自动轮换机制，避免单 Key 配额耗尽
- 一键跳转到各平台申请页面

### 📊 数据查询
- 关键词搜索已采集的 POI 数据
- 按平台筛选
- 支持精确/前缀/包含三种匹配模式
- 列表/地图/分屏三种视图模式

### 📤 数据导出
- **Excel** (.xlsx) - 表格格式
- **JSON** - 结构化数据
- **MySQL** (.sql) - 数据库脚本
- 按地区筛选导出范围
- 导出前数据预览

### 🎨 界面特性
- 现代化 UI 设计 (shadcn/ui)
- 亮色/暗色/跟随系统主题切换
- 实时采集日志显示
- 采集进度可视化

## 快速开始

1. 下载并安装应用
2. 打开设置 (右上角⚙️) → API Key → 添加各平台 Key
3. 设置 → 地区管理 → 选择采集地区
4. 进入采集页面 → 选择类别 → 开始采集
5. 查询页面搜索数据 / 导出页面导出数据

## 技术栈

- **框架**: Tauri 2.x (Rust + WebView)
- **前端**: React 18 + TypeScript + Vite
- **UI**: shadcn/ui + Tailwind CSS
- **数据库**: SQLite
- **地图**: Leaflet
